# 🤖 Build Nuelz Forex AI Pro X

You are a senior Python software engineer, AI engineer, quantitative analyst, UI/UX designer, and Telegram bot developer.

Build a complete production-ready application called **Nuelz Forex AI Pro X**.

Nuelz Forex AI Pro X is a private AI-powered Forex market analysis assistant that continuously monitors the Forex market, performs advanced technical analysis, identifies high-confidence trading opportunities, and delivers beautifully formatted market analysis through Telegram and a modern web dashboard.

The application must NEVER claim guaranteed profits or guaranteed winning trades.

The application provides professional market analysis and alerts only.
